# Pokemon and Friends Matching Game

A simple, fun matching game featuring Pokemon and their friends.

## How to Play

1. Open `index.html` in your web browser.
2. Click on a card to reveal the image.
3. Click on a second card to see if it's a match.
4. If the cards match, they will stay face up.
5. If they don't match, they will be flipped back over.
6. The game is over when all the cards have been matched.
7. Click the "Reset Game" button to play again.
